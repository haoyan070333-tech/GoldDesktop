﻿# 黄金/积存金走势监测与预测桌面程序（V1.0）

【程序核心】一款适配Windows 64位（Win10/11）的独立exe程序，无需安装.NET运行时，可实时监测现货黄金价格、展示短期走势并预测未来30分钟区间，还支持价格提醒，助力快速掌握黄金行情。
【核心功能】

✅ 实时监测：双单位切换（美元/盎司、人民币/克），每15秒刷新一次最新价格及涨跌幅，同步展示最近15分钟价格折线图。

✅ 走势预测：每2分钟执行一次预测，含两点核心模块——线性通道预测（三点一线回归+标准差带宽，附偏多/偏空指导）、Monte Carlo随机模拟（给出中值、上下沿区间及3条典型走势线）。

✅ 价格提醒：支持设置“低于/高于”目标价，触发时自动循环播放提示音+弹窗警告，关闭弹窗即停止提醒，适配当前显示单位。

✅ 便捷操作：图表支持滚轮缩放、右键平移、左键框选放大；可锁定横坐标，方便聚焦特定历史区间盯盘。

【运行须知】

• 环境要求：Windows 10/11 64位系统，需能访问网络获取行情数据。

• 启动方式：找到发布目录中的GoldDesktop.exe双击启动，建议复制至常用位置并创建桌面快捷方式。

• 界面布局：默认全屏，左列显示时间、单位切换、价格、提醒设置及预测列表，右列展示折线图、预测扇形通道及典型走势。

⚠️ 重要提示：所有预测结果仅为统计模拟，不构成任何交易建议。
